# DESIGN.md

## 产品定位
赛博朋克风格 AI 实时运动教练 — 3D 怪物模型 + 骨架检测 + LLM 智能分析 + TTS 语音反馈 + 语音交互。

## 气质与意象
赛博朋克健身房，霓虹灯管映在钢化玻璃上，深色空间里浮动着青色/橙色数据流。3D 怪物随运动强度变化而切换形态。UI 像高科技作战指挥台——暗色底板、荧光数据、呼吸感的指示灯。运动者一眼扫到关键反馈，沉浸在游戏化训练体验中。

## 配色方案
- 主背景：`#0a0e17`（深渊蓝黑）
- 卡片/面板：`#111827` / `#1a1f2e`
- 主强调色-青：`#06b6d4`（数据流/安全/连接）
- 次强调色-橙：`#f97316`（心率/警告/活跃）
- 危险色：`#ef4444`（错误/纠正）
- 成功色：`#10b981`（正确/完成）
- 文字主色：`#e2e8f0`
- 文字次色：`#94a3b8`
- 怪物色：7种运动各对应独特配色（绿/红/黄/紫/青/橙/玫红）

## 字体排版
- 英文/数字：monospace（数据感，JetBrains Mono / Consolas）
- 中文/UI：系统默认 -apple-system, "PingFang SC", "Microsoft YaHei"

## 动效与交互
- 骨架连线：平滑插值，绿色=OK/橙色=需调整/红色=错误
- 3D 怪物：随运动强度自动切换形态（自动模式），或手动选择
- 教练反馈文字：打字机渐入
- 语音反馈：TTS 合成后自动播放
- 动作计数：数字变化动效
- 里程碑：canvas-confetti 庆祝特效（5/10/15/20/30/50次）
- 连接状态：呼吸灯脉冲动画
- 背景网格：微妙的网格线动画

## 页面结构
- 左侧 ~60%：3D 怪物模型 (Spline) + 摄像头画面 + 骨架叠加
- 右侧 ~40%：统计数据行 + 教练反馈面板 + 语音交互 + 运动控制栏
- 底部：7种运动模式切换（深蹲/俯卧撑/弓步蹲/平板支撑/开合跳/高抬腿/自定义）
- 教练人格选择（7种风格：铁血教官/佛系禅师/毒舌损友等）

## 设计禁忌
- 不要用大面积纯白背景
- 不要弹窗打断运动流程（训练总结弹窗除外，仅在结束时触发）
- 不要密集文字超过3行
- 不要失去赛博朋克暗色基调
