# DESIGN.md

## 产品定位
实时运动搭子教练 — 支持本地摄像头/树莓派远程双模式，云端骨架检测 + LLM 智能分析 + TTS 语音反馈。

## 气质与意象
赛博朋克健身房 — 霓虹灯光打在暗色镜面上，Spline 3D 怪兽随运动强度变形。数据面板像飞行仪表盘，信息密度高但不压迫。教练反馈通过3D怪兽嘴部动画+语音同步呈现。

## 配色方案
- 主背景：深黑蓝 `#060B15` (cyber-dark)
- 卡片/面板：`#1A1D27`（毛玻璃叠加）
- 主强调色：霓虹青 `#00E5FF` (cyber-cyan)
- 警告色：珊瑚红 `#FF4757`
- 安全色：薄荷绿 `#22D3A7`
- 文字主色：`#E8E9ED`
- 文字次色：`#8B8FA3`

## 字体排版
- 英文/数字：等宽字体 (font-mono)
- 中文/UI：系统默认 -apple-system, "PingFang SC", "Microsoft YaHei"

## 动效与交互
- 扫描线覆盖 (scanlines)
- 3D怪兽嘴部动画 (Spline + requestAnimationFrame)
- 教练反馈面板：毛玻璃 + 霓虹辉光
- 分数变化 → 彩纸特效 (canvas-confetti)
- 训练完成 → 总结弹窗 + 彩纸
- 心跳图标动画

## 页面结构
- 左侧 1/4：AI教练面板 (3D怪兽 + 教练消息 + 怪兽选择器 + 人设/语音定制)
- 右侧 3/4：数据仪表盘 + 摄像头/骨架视图
  - 顶部：连接状态 + 语音开关 + 模式切换
  - 中部：StatsRow (动作/心率/次数/分数)
  - 主体：摄像头+骨架叠加 + 运动选择器 + 开始/结束按钮
- 弹窗：AI定制计划 / 训练总结

## 设计禁忌
- 不要用蓝紫色渐变
- 不要大面积纯白背景
- 不要弹窗打断运动流程
- 不要密集文字超过3行
