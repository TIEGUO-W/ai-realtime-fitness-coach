# DESIGN.md

## 产品定位
实时运动搭子教练 — 赛博朋克健身房，3D怪兽随运动变形，数据面板像飞行仪表盘。

## 气质与意象
午夜赛车游戏仪表盘 — 黑色碳纤维面板上嵌着霓虹仪表，数据像赛车HUD一样醒目但不压迫。3D怪兽是副驾驶，跟你一起练。整体感觉是"在赛博健身房里跟一只电子宠物一起流汗"。

## 配色方案
- 主背景：`#05080F` (极深黑蓝)
- 面板/卡片：`#0C1018` (碳纤维暗层)
- 卡片悬浮：`#111827` (微微亮起)
- 主强调色：霓虹青 `#00E5FF`
- 次强调色：烈焰橙 `#FF6B35`
- 警告色：珊瑚红 `#FF4757`
- 安全色：薄荷绿 `#22D3A7`
- 文字主色：`#F0F1F5`
- 文字次色：`#6B7280`
- 边框色：`rgba(0,229,255,0.08)`

## 字体排版
- 数据/数字：JetBrains Mono 或 font-mono (tabular-nums)
- 中文/UI：系统默认 PingFang SC / Microsoft YaHei
- 标题：大写字母间距 0.15em，font-bold
- 数据值：text-2xl/3xl font-mono font-bold
- 数据标签：text-[10px] uppercase tracking-widest text-slate-500

## 设计规范

### 卡片系统
- 圆角：rounded-xl (12px)
- 边框：1px solid rgba(0,229,255,0.06)
- 背景：bg-[#0C1018] / bg-[#111827]
- 悬浮：hover:border-cyber-cyan/20 hover:bg-[#111827]
- 阴影：不用大阴影，用微妙的边框发光
- 间距：gap-3, p-4

### 数据仪表卡片
- 4列等宽网格
- 顶部：10px大写标签 (text-[10px] tracking-widest text-slate-500)
- 中间：大号数据 (text-2xl font-mono font-bold)
- 底部：进度条或辅助信息
- 状态色：正常=cyber-cyan, 警告=orange, 危险=red, 优秀=green

### 教练面板
- 气泡对话：用户右对齐(青色边框), 教练左对齐(暗灰底)
- 最多显示3条
- 教练头像用3D怪兽缩略图
- 消息文字 14px, 行高 1.5

### 摄像头区域
- 圆角卡片，深色背景
- 四角装饰用简洁的L型线
- 角落信息标签：BACKEND LIVE, REC, 动作名
- 运动选择器：底部浮层药丸按钮
- 开始/结束按钮：底部右侧，大圆角药丸

### 3D怪兽区域
- 占左侧面板上半部分
- 底部发光环随质量变色
- 怪兽名在底部居中
- 右上角切换按钮 + 模式指示灯

### 状态栏
- 极简一行：连接状态 + 语音开关 + 骨架检测 + 计时器
- 背景半透明

### 音乐控制
- 左侧面板底部
- 极简：🎵 + 曲目名 + 播放/暂停
- 展开选择列表

## 动效规范
- 卡片hover：border-color transition 200ms
- 数据变化：数字用 tabular-nums，变化时微闪
- 按钮hover：发光强度增加 + 轻微scale
- 怪兽：bounce(次数+1), shake(分数骤降), pulse(变形警告)
- 扫描线：极淡(0.03透明度)，不干扰阅读

## 设计禁忌
- 不要蓝紫色渐变
- 不要大面积纯白
- 不要弹窗打断运动
- 不要超过3行密集文字
- 不要用粗重的阴影(用微妙发光代替)
- 不要用纯色块分隔(用极淡边框)
- 数据标签不要用emoji，只用图标
